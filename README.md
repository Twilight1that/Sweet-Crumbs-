# Sweet Crumbs Bakery - Luxury Artisan Experience

Welcome to **Sweet Crumbs Bakery**, a premium luxury bakery website that represents the pinnacle of artisanal baking excellence. This multi-page website has been transformed into a high-end brand experience suitable for a ₹1,00,000+ professional bakery business.

## 🎨 Design Philosophy

- **Dark Elegant Theme**: Deep brown/near black background with gold accent color (#d4af37)
- **Premium Typography**: Playfair Display for headings, Poppins for body text
- **Luxury Layout**: Centered max-width (1200px), 80px section spacing, symmetrical alignment
- **High-End Animations**: Smooth transitions, fade-in effects, hover transformations

## 🏗️ Project Structure

```
sweet-crumbs-bakery/
├── index.html           # Luxury homepage with hero, menu preview, reviews, contact
├── about.html           # Artisan story, team profiles, awards
├── menu.html            # Premium menu with categorized products
├── contact.html         # Elegant contact form with map integration
├── css/
│   └── style.css        # Complete luxury styling with CSS variables
├── js/
│   └── script.js        # Advanced animations and interactions
├── images/              # Directory for high-quality bakery images
└── README.md           # This documentation
```
